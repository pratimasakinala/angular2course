Built-in Directives Recap

  • What is a directive?
  • Two types of directives?
  • Some of basic built-in directives?
  • Practice:
    • ngIf-
    • Create a property displayParagraph of type boolean and set it to false by default.
    • Display a paragraph with any text of your choice.
    • Add a condition (ngIf) on the above paragraph to check the value of displayParagraph.
    • The paragraph should be displayed if displayParagraph is true.
    • Now set the value of displayParagraph to true and observe the change in template.

    • ngStyle-
    • Use ngStyle to set
      • position: absolute;
        right: 0;
        top: 0;
      properties of the above paragraph.

    • ngClass-
    • Create a css class color-red that sets the color to red.
    • Create another css class background-yellow that sets the background-color to yellow.
    • Create another css class hide-text that sets the display property to none.
    • Add a new paragraph with any text below the existing paragraph.
    • Add color-red and background-yellow classes using string assignments to ngClass.
    • Create another paragraph with any text and add color-red and background-yellow classes using array assignemnt to ngClass.
    • Create another paragraph with some text and add color-red and background-yellow classes using object assignment to ngClass. To this paragraph, add hide-text class if displayParagraph is true.

    • ngFor-
    • Create a property users which is an array of objects.
    • Each user object consists of a name and age property.
    • Create some dummy data in the users array. (create a class maybe? or not.)
    • Display each user's details (name and age used in proper sentence) as a list item in an unordered list under the previously added paragraphs in the template.
