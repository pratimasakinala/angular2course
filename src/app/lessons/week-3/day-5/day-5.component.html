Practicing Databinding

Practice 1 (Passing Data to nested component):

  • Create two components: ParentComponent (manually) and ChildComponent (using CLI).
  • In ParentComponent, add a header (h1) with text "I'm a Container Component".
  • Display ChildComponent below this header.
  • In ChildComponent, create a property title and set it to "I'm a nested component".
  • Display a header (h2) with the value of title in ChildComponent template (use inline template).
  • Run the project to make sure the components load with proper data.
  • Modify ChildComponent such that the title property is no longer hard-coded to the string value. But it takes any value passed into it from ParentComponent.
  • Create a childTitle property in ParentComponent and set it to "Text passed to child".
  • Update the ParentComponent template to bind to title property of ChildComponent and pass in childTitle property into it.

Practice 2 (Passing Data from nested component):

  • Continue working on the same project that was created in Practice 1 above.
  • Add a button in ChildComponent below the header with text "Click me!".
  • On clicking this button, execute a method notifyParent that emits the event to ParentComponent.
  • In ParentComponent template, listen to the event (created in ChildComponent) and execute a method onNotify (which prints the message in console).
